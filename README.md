# [BurnStyle](https://burn-style.vercel.app/)

> **Burn** your money, show your **Style**.

お金の使い方には、その人の個性が宿る。

## コンセプト

収入は社会が決める。だが支出は自分が決める。

給与や報酬は市場や組織の論理で決まる。そこに個性の入り込む余地はほとんどない。
一方、同じ手取りでも何に使うかは人によってまるで違う。コーヒーに月1万円使う人と、本に月1万円使う人は、同じ金額でもまったく別の人生を歩いている。

支出とは、日々繰り返される小さな意思決定の集積。
何を買い、何を買わないか。その選択の連なりが、言語化されない価値観を静かに映し出す。

「節約」ではなく「理解」。
BurnStyleは支出を減らすためのツールではない。自分が何に価値を感じているのかを知るためのツール。
削るべき無駄を見つけるのではなく、自分だけの消費の輪郭を眺める。

お金の燃やし方に、あなたのスタイルが宿る。
