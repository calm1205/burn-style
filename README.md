# [BurnStyle](https://burn-style.vercel.app/)

> **Burn** your money, show your **Style**.

お金の使い方には、その人の個性が宿る。

## コンセプト

「何にお金を使っているか」に特化した支出記録アプリ。
収入や資産管理は扱わず、支出パターンからあなたの価値観や生活スタイルを可視化する。

## 特徴

- **シンプルな記録** - 名前・金額・日時・カテゴリだけ。ワンタップで支出を記録
- **パスワードレス** - パスキー（Face ID / Touch ID）でログイン
- **多角的な可視化** - 月別・年別トレンド、カテゴリ別割合、ヒートマップ、バブルチャート

## やらないこと

- 収入管理・資産管理・口座連携
- 予算設定・アラート
- チーム・家族での共有
- 広告表示
