# 支出管理

家計簿の中核機能。支出の記録・集計・サブスクリプション管理。

## 概要

- 支出（出費）のみを管理対象とする
- 入力項目は最小限: タイトル・金額・カテゴリ
- サブスクリプションはテンプレート方式で管理

## 支出記録

### 入力項目

| 項目 | 必須 | 説明 |
|------|------|------|
| タイトル | ○ | 支出の内容（例: スーパーで買い物） |
| 金額 | ○ | 支払金額（円、正の整数） |
| カテゴリ | ○ | カテゴリから複数選択可能 |

- 日付はデフォルトで当日、変更可能
- 記録後の編集・削除が可能（論理削除）

### カテゴリ

- カテゴリはユーザーが自由に作成・編集・削除可能（詳細は [カテゴリ管理](category.md) を参照）

#### 初期カテゴリ（seed）

| カテゴリ | 例 |
|----------|-----|
| 食費 | スーパー、外食、コンビニ |
| 日用品 | 洗剤、ティッシュ |
| 交通費 | 電車、バス、タクシー |
| 住居費 | 家賃、管理費 |
| 光熱費 | 電気、ガス、水道 |
| 通信費 | スマホ、インターネット |
| 娯楽費 | 映画、ゲーム、書籍 |
| 医療費 | 病院、薬 |
| 衣服費 | 衣類、靴 |
| その他 | 上記に該当しないもの |

- 上記は初期データとして提供。ユーザーが自由に追加・変更可能

## サブスクリプション（テンプレート方式）

定期支払いを「テンプレート」として登録し、手動で一括記帳する方式。

### テンプレート登録

| 項目 | 必須 | 説明 |
|------|------|------|
| タイトル | ○ | サービス名（例: Netflix） |
| 金額 | ○ | 月額料金 |
| カテゴリ | ○ | カテゴリを選択 |

- テンプレートの編集・削除が可能

### 一括記帳

- サブスクリプション一覧画面から「今月分を記帳」ボタンを押下
- 登録済みテンプレートをもとに支出レコードを一括生成
- 記帳対象のテンプレートを個別に選択可能（チェックボックス）
- 記帳済みかどうかは月単位で判定

### 制約

- 自動記帳（cron等）は行わない
- 年払い等の頻度設定は将来検討

## 集計・表示

### 期間別集計

- 指定した期間内の支出合計を表示
- フィルタ: 月単位（デフォルト: 当月）

### カテゴリ別集計

- カテゴリごとの合計金額を表示
- 期間フィルタと組み合わせ可能

### 将来検討

- グラフ表示（円グラフ・棒グラフ）
- 前月比較
- 予算設定・超過アラート

## 制約・注意事項

- 通貨は日本円（JPY）のみ
- 収入管理は対象外
- オフライン対応は対象外
