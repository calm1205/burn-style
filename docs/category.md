# カテゴリ管理

支出の分類に使用するカテゴリの作成・編集・削除。

## 概要

- ユーザーごとに独立したカテゴリセットを保持
- 1つの支出に複数カテゴリを関連付け可能

## 操作

### カテゴリ作成

- カテゴリ名を入力して作成

### カテゴリ編集

- インラインエディットで名前を変更
- 編集アイコンをクリックするとテキスト入力に切替

### カテゴリ削除

- 削除アイコンをクリックして削除

## 初期カテゴリ

seedデータとして以下が登録:

| カテゴリ | 例 |
|----------|-----|
| 食費 | スーパー、外食、コンビニ |
| 日用品 | 洗剤、ティッシュ |
| 交通費 | 電車、バス、タクシー |
| 住居費 | 家賃、管理費 |
| 光熱費 | 電気、ガス、水道 |
| 通信費 | スマホ、インターネット |
| 娯楽費 | 映画、ゲーム、書籍 |
| 医療費 | 病院、薬 |
| 衣服費 | 衣類、靴 |
| その他 | 上記に該当しないもの |

- ユーザーが自由に追加・変更・削除可能

## API

| エンドポイント | 説明 |
|---------------|------|
| `GET /categories` | カテゴリ一覧取得 |
| `POST /categories` | カテゴリ作成 |
| `PATCH /categories/{uuid}` | カテゴリ名更新 |
| `DELETE /categories/{uuid}` | カテゴリ削除 |

## 制約・注意事項

- ユーザー単位で管理（他ユーザーのカテゴリは参照不可）
- 存在しないカテゴリへのアクセスは `404 Not Found`
