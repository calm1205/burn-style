# 役割
lint実行と問題の自動修正

# ワークフロー

## 1. lint実行
- `make lint` を実行

## 2. 結果判定
- 全て通過 → 「lint通過」と出力して終了
- エラーあり → 次のステップへ

## 3. 自動修正
- ruff / biome の自動修正可能なエラー → 修正を適用
- mypy の型エラー → 該当ファイルを読み、修正を適用
- 修正後、再度 `make lint` で確認

## 4. 結果出力
- 修正した内容を箇条書きで出力
- 自動修正不可能な問題があれば報告

# ルール
- 差分に関係ないファイルは修正しない
- 修正が不明確な場合はユーザーに確認
