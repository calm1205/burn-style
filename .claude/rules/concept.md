# プロダクトコンセプト

## ビジョン

**支出だけを記録し、ユーザーの個性を可視化する**

収入や資産の管理は行わず、「何にお金を使ったか」のみに集中する。
支出パターンを通じて自身の価値観・ライフスタイルを客観的に理解する。

## 対象ユーザー

- 個人利用 (組織・チーム向けではない)
- 自身の支出習慣を把握したい人
- 複雑な家計簿アプリを続けられなかった人

## デザイン哲学

### 直感的に操作できるシンプルな UI

- 入力項目は最小限 (名前・金額・日付・カテゴリ)
- ワンタップで支出を記録
- パスキーによるパスワードレス認証でログイン摩擦を排除

### 可視化による気づき

- 月次・年次のトレンドチャート
- カテゴリ別の支出内訳
- 日別支出パターンのヒートマップ
- 金額帯分布のバブルチャート

### スコープ外

- 収入管理
- 資産管理・銀行口座連携
- 予算設定・アラート
- チーム・家族共有
- 広告
