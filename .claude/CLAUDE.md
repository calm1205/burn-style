# Finance - プロジェクト規約

## 概要
- 家計管理アプリ（フルスタック）
- バックエンド: FastAPI + SQLAlchemy + PostgreSQL
- フロントエンド: React + Vite + Tailwind CSS v4
- 認証: WebAuthn/パスキー + JWT
